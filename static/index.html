Updating APK-JTM
If you installed via Git (recommended)
  1. Open a terminal in the APK-JTM folder
  2. Run: git pull
  3. Re-run the launcher — it will update dependencies automatically
If you downloaded a ZIP
  1. Download the new release from GitHub Releases
  2. Extract anywhere and run the launcher
Your config and reports are stored outside the app folder — they are not affected by an update and no files need to be copied. You can see exactly where in ⚙ Settings.
